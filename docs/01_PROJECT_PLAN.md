# 01. 프로젝트 플랜 (Integrated Project Plan & Roadmap)

## 0. 프로젝트 개요
- **앱 이름**: 오늘의 짝꿍 (Today's Match)
- **비전**: 단순한 운세를 넘어, 사용자의 입체적인 성향(사주, 휴먼디자인, 에니어그램)을 분석하여 "나"를 발견하고 "최고의 케미"를 찾아주는 엔터테인먼트 플랫폼.
- **타깃**: 토스 인앱을 사용하는 20~40대 성인 사용자.

---

## 2. 개발 로드맵 (Phases)

### Phase 1: 기반 다지기 (Foundation) - ✅ 완료
- [x] v1.0.0 런칭 (안정적인 운세 확인 및 심화 리포트)
- [x] 토스 인앱 딥링크 및 업데이트 프로세스 확립
- [x] 통합 프로필 로직(사주/휴디/에니어) 기본형 탑재

### Phase 2: 나를 더 깊이 알기 (Deep Personalization)
- [ ] **데이터 페칭 자동화**: `bazi-calculator`, `hdkit` 등을 활용한 사주/휴디 자동 도출.
- [ ] **LLM 기반 심층 분석**: 프롬프트 엔지니어링을 통한 고도화된 연애 조언.
- [ ] **보상형 광고(Rewarded Ads)**: 광고 시청 시 심층 리포트 해제 로직 완성.

### Phase 3: 시각화 & 캐릭터 (Visual & Engagement)
- [ ] **나만의 캐릭터 생성**: AI 이미지 생성 기술을 이용한 비주얼 카드 발급.
- [ ] **오늘의 짝꿍 시각화**: 오늘 나랑 딱 맞는 가상 짝꿍 이미지 제공.
- [ ] **도감 시스템**: 매일 다른 캐릭터 카드를 수집하는 재미 요소 추가.

### Phase 4: 현실 연결 (Social & Real Connection)
- [ ] **친구/연인 매칭**: 지인 프로필 등록 및 매일 아침 "베스트 케미" 알림.
- [ ] **궁합 배틀**: 내 캐릭터 vs 친구 캐릭터의 시뮬레이션 기능.
