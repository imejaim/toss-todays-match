## 📱 현재 상태

### 샌드박스 vs QR 화면 차이 이유
| 항목 | 샌드박스 | QR (토스앱) |
|---|---|---|
| 프로필 상태 | 입력 안됨 | 이미 입력됨 ("초이") |
| 표시 내용 | 기본 화면만 | 전체 기능 표시 |
| **원인** | LocalStorage가 비어있음 | 기존 저장 데이터 있음 |

**➡️ 샌드박스에서 프로필 입력하면 QR과 동일하게 표시됩니다.**
