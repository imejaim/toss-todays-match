## 🔐 Git Stash 사용 주의사항

> ⚠️ **경고**: `git stash`는 임시 저장이며, 영구 보존이 아닙니다!

### Stash 사용 시 체크리스트
1. [ ] stash 전 변경 내용 메모 (어떤 기능인지)
2. [ ] stash 후 `git stash list`로 확인
3. [ ] 작업 완료 후 **반드시** `git stash pop` 또는 `git stash apply`
4. [ ] 복구 후 정상 동작 확인
5. [ ] 필요 없으면 `git stash drop`으로 명시적 삭제

### 위험한 작업 시
```bash
# stash 내용 확인
git stash show -p stash@{0}

# stash 복구 (삭제하지 않음)
git stash apply

# stash 복구 + 삭제
git stash pop
```
