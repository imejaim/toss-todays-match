# 3. 휴먼디자인(Human Design) 기초 이론

## 1. 개요
휴먼디자인 시스템은 고대의 지혜와 현대 과학(유전학, 양자물학 등)을 통합하여 개인의 고유한 에너지 메커니즘을 분석하는 도구입니다. 본 시스템은 이를 통해 사용자의 효율적인 의사결정 전략과 행동 양식을 제안합니다.

## 2. 핵심 분석 요소 (Key Elements)

### 2.1 타입 (Type) - "에너지 운영 방식"
*   **매니패스터 (Manifestor)**: 영향력을 미치고 시작하는 자.
*   **제너레이터 (Generator)**: 반응하고 지속적으로 에너지를 창조하는 자.
*   **프로젝터 (Projector)**: 타인의 에너지를 가이드하고 조율하는 자.
*   **리플렉터 (Reflector)**: 환경의 상태를 비추고 관찰하는 자.

### 2.2 전략 (Strategy) - "올바른 행동 지침"
각 타입이 저항 없이 삶을 살아가기 위한 핵심 지침입니다. (예: 매니패스터-알리기, 제너레이터-반응하기)

### 2.3 내적 권위 (Inner Authority) - "의사결정 도구"
머리가 아닌 몸의 반응으로 올바른 결정을 내리는 지점입니다. (예: 감정 권위, 천골 권위 등)

### 2.4 프로필 (Profile) - "인생의 역할"
내가 삶의 무대에서 어떤 의상을 입고 어떤 역할을 수행하는지 나타내는 6개 라인의 조합입니다 (예: 1/3, 3/5 등).

## 3. 시각화 가이드
*   **BodyGraph**: 주요 센터(Center)의 정의 여부를 색상으로 표현.
*   **Strategy Badge**: 현재 사용자가 취해야 할 전략을 아이콘(번개, 확성기 등)으로 표시.
*   **Character Item**: 프로필과 타입의 특성을 캐릭터의 소품(왕관, 안경, 가방 등)으로 반영.
