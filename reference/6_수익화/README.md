# 수익화 (Monetization)

앱인토스(Apps in Toss)에서 비즈니스를 확장하고 수익을 창출하는 방법을 안내합니다.

## 주요 수익화 모델

### 1. 인앱 광고 (In-app Ads)
- [인앱 광고 문서 바로가기](https://developers-apps-in-toss.toss.im/ads/intro.html)
- 앱 내에 광고를 탑재하여 광고 수익을 창출합니다.
- 배너 광고, 리워드 광고 등 다양한 형태를 지원합니다.

### 2. 인앱 결제 (In-app Payments)
- [인앱 결제 문서 바로가기](https://developers-apps-in-toss.toss.im/iap/intro.html)
- 디지털 상품(아이템, 구독권 등)을 쉽고 안전하게 판매할 수 있습니다.
- 토스 결제 시스템과 연동되어 매끄러운 구매 경험을 제공합니다.

### 3. 토스페이 (Toss Pay)
- [토스페이 문서 바로가기](https://developers-apps-in-toss.toss.im/tosspay/intro.html)
- 원탭 결제 경험을 통해 결제 이탈율을 줄이고 전환율을 높입니다.
- 빠르고 간편한 결제 수단을 앱에 도입하세요.

---
*참고: 이 문서는 [앱인토스 개발자 센터 - 수익화](https://developers-apps-in-toss.toss.im/revenue/overview) 내용을 기반으로 정리되었습니다.*
